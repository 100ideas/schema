Cite

Status: unstable, Role: secondary

A reference to a CreativeWork that is cited in another CreativeWork.

Properties

target (required) string The target of the citation (URL or reference ID). Cite
type (required) enum<​Cite​> The name of the type and all descendant types. Entity
citationMode enum<​normal | suppressAuthor​> How the cite is rendered in the surrounding text. Cite
content array<​InlineContent​> Optional structured content/text of this citation. Cite
id string The identifier for this item. Entity
meta object Metadata associated with this item. Entity
pageEnd string | integer The page on which the work ends; for example "138" or "xvi". Cite
pageStart string | integer The page on which the work starts; for example "135" or "xiii". Cite
pagination string Any description of pages that is not separated into pageStart and pageEnd; for example, "1-6, 9, 55". Cite
prefix string A prefix to show before the citation. Cite
suffix string A suffix to show after the citation. Cite

A Cite node is used within a CreativeWork, usually an Article, to refer to an other CreativeWork. Often a Cite will be associated with other citations, in a CiteGroup.

Examples

The following example of a (very) short article, shows how a Cite is used to refer to another Article in the article's references.

{
  "type": "Article",
  "title": "An example of using the Cite node type",
  "authors": [
    {
      "type": "Person",
      "givenNames": ["Joe"],
      "familyNames": ["Bloggs"]
    }
  ],
  "content": [
    {
      "type": "Paragraph",
      "content": [
        "A citation of ",
        {
          "type": "Cite",
          "target": "some-one-else-1991"
        },
        "."
      ]
    }
  ],
  "references": [
    {
      "type": "Article",
      "id": "some-one-else-1991",
      "title": "Another article by someone else",
      "authors": [
        {
          "type": "Person",
          "givenNames": ["Some", "One"],
          "familyNames": ["Else"]
        }
      ],
      "datePublished": "1991"
    }
  ]
}

Encodings

---
title: An example of using the Cite node type
authors:
  - type: Person
    #...
references:
  - type: Article
    id: some-one-else-1991
    #...
---

A citation of @some-one-else-1991.
