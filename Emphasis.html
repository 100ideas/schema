Emphasis

Status: stable, Role: secondary

Emphasised content.

Properties

content (required) array<​InlineContent​> The content that is marked. Mark
type (required) enum<​Emphasis​> The name of the type and all descendant types. Entity
id string The identifier for this item. Entity
meta object Metadata associated with this item. Entity

Examples

To illustrate how Emphasis nodes are represented in alternative formats, we'll use the following example, in context, within a Paragraph:

{
  "type": "Paragraph",
  "content": [
    "The following content has extra ",
    {
      "type": "Emphasis",
      "content": ["emphasis"]
    }
  ]
}

Encodings

This section describes common encodings for this node type. These samples are generated from the above examples by Encoda, but you can also author them in each format.

HTML

HTML natively supports Emphasis nodes with the <em> element.

<p>The following content has extra <em>emphasis</em></p>

JATS

The JATS equivalent to Emphasis is the <italic> element.

<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE article PUBLIC "-//NLM//DTD JATS (Z39.96) Journal Archiving and Interchange DTD v1.1 20151215//EN" "JATS-archivearticle1.dtd">
<article xmlns:xlink="http://www.w3.org/1999/xlink" article-type="research-article">
    <front>
        <title-group>
            <article-title/>
        </title-group>
        <contrib-group/>
    </front>
    <body>
        <sec>The following content has extra 
            <italic>emphasis</italic>
        </sec>
    </body>
    <back>
        <ref-list>
            <title>References</title>
        </ref-list>
    </back>
</article>

YAML

type: Paragraph
content:
  - 'The following content has extra '
  - type: Emphasis
    content:
      - emphasis

Markdown

Emphasis in Markdown can be achieved with underscores (_) or asterisks (*). See also the MDAST reference.

The following content has extra _emphasis_

Pandoc

The equivalent of Emphasis in Pandoc is the Emph element. The above example in Pandoc JSON:

{
  "blocks": [
    {
      "t": "Para",
      "c": [
        {
          "t": "Str",
          "c": "The following content has extra "
        },
        {
          "t": "Emph",
          "c": [
            {
              "t": "Str",
              "c": "emphasis"
            }
          ]
        }
      ]
    }
  ],
  "pandoc-api-version": [
    1,
    17,
    5,
    4
  ],
  "meta": {}
}
