Link

The Link schema represents a hyperlink to other pages, sections within the same document, resources, or any URL.

A hyperlink to other pages, sections within the same document, resources, or any URL.

Properties

content (required) array<​InlineContent​> The textual content of the link. Link
target (required) string<uri> The target of the link. Link
type (required) enum<​Link​> The name of the type and all descendant types. Entity
id string The identifier for this item. Entity
meta object Metadata associated with this item. Entity
relation string The relation between the target and the current thing. Link
title string A title for the link. Link

Examples

This is a simple Link that will be encoded to different formats as an example. 

{
  "type": "Link",
  "content": ["Stencila’s website"],
  "target": "https://stenci.la"
}

Encodings

This section describes common encodings for this node type. These samples are generated from the above examples by Encoda, but you can also author them in each format.

HTML

Link is analogous to the HTML <a> element.

<a href="https://stenci.la">Stencila’s website</a>

JATS

Link is analogous to the JATS <ext-link> element.

<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE article PUBLIC "-//NLM//DTD JATS (Z39.96) Journal Archiving and Interchange DTD v1.1 20151215//EN" "JATS-archivearticle1.dtd">
<article xmlns:xlink="http://www.w3.org/1999/xlink" article-type="research-article">
    <front>
        <title-group>
            <article-title/>
        </title-group>
        <contrib-group/>
    </front>
    <body>
        <sec>Stencila’s website</sec>
    </body>
    <back>
        <ref-list>
            <title>References</title>
        </ref-list>
    </back>
</article>

Markdown

Link is analogous to the mdast Link node.

[Stencila’s website](https://stenci.la)

OpenDocument

Link is analogous to the OpenDocument <text:a> element. This odt file was generated from the above example.
