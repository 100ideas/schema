QuoteBlock

Status: stable, Role: secondary

A section quoted from somewhere else.

Properties

content (required) array<​BlockContent​> The content of the quote. QuoteBlock
type (required) enum<​QuoteBlock​> The name of the type and all descendant types. Entity
cite Cite | string<uri> The source of the quote. QuoteBlock
id string The identifier for this item. Entity
meta object Metadata associated with this item. Entity

The QuoteBlock schema represents an extended quoted sections.

Examples

Simple

{
  "type": "QuoteBlock",
  "content": [
    {
      "type": "Paragraph",
      "content": [
        "If you wish to make an apple pie from scratch, you must first invent the universe.",
        "by Carl Sagan - Cosmos"
      ]
    }
  ]
}
