TableRow

Status: experimental, Role: secondary

A row within a Table.

Properties

cells (required) array<​TableCell​> An array of cells in the row. TableRow
type (required) enum<​TableRow​> The name of the type and all descendant types. Entity
id string The identifier for this item. Entity
meta object Metadata associated with this item. Entity
rowType enum<​header | footer​> If present, indicates that all cells in this row should be treated as header cells. TableRow

A TableRow type is primarily a container for a list of Table Cell types.

Examples

Simple

This is the most basic form of TableRow you can have, only requiring the content field.

{
  "type": "TableRow",
  "cells": [
    {
      "type": "TableCell",
      "content": [1]
    }
  ]
}

An Empty Row

A TableRow can be empty and still valid, representing a table row with no data. The content field however, is still required.

{
  "type": "TableRow",
  "content": []
}
