Paragraph

Status: experimental, Role: secondary

Paragraph

Properties

content (required) array<​InlineContent​> The contents of the paragraph. Paragraph
type (required) enum<​Paragraph​> The name of the type and all descendant types. Entity
id string The identifier for this item. Entity
meta object Metadata associated with this item. Entity

The Paragraph schema represents a paragraph, or a block of text. It can contain any valid InlineContent nodes.

Examples

Simple

{
  "type": "Paragraph",
  "content": ["Some text content representing ideas expressed as words."]
}

Nested Content

{
  "type": "Paragraph",
  "content": [
    "Some text with some",
    {
      "type": "Emphasis",
      "content": ["emphasised words"]
    },
    " and ",
    {
      "type": "Strong",
      "content": ["some strongly emphasised words"]
    }
  ]
}
