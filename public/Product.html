Product

Status: stable, Role: tertiary

Any offered product or service. For example, a pair of shoes; a haircut; or an episode of a TV show streamed online.

Properties

type (required) enum<​Product​> The name of the type and all descendant types. Entity
alternateNames array<​string​> Alternate names (aliases) for the item. Thing
brands array<​Brand​> Brands that the product is labelled with. Product
description string | array<​Node​> A description of the item. Thing
id string The identifier for this item. Entity
logo string<uri> | ImageObject The logo of the product. Product
meta object Metadata associated with this item. Entity
name string The name of the item. Thing
productID string Product identification code. Product
url string<uri> The URL of the item. Thing

The Product type allows you to provide details about a product such as the product brand, logo and ID. This type can be used as any kind of product that is not CreativeWork item.
