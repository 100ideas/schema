Strong

Status: stable, Role: secondary

Strongly emphasised content.

Properties

content (required) array<​InlineContent​> The content that is marked. Mark
type (required) enum<​Strong​> The name of the type and all descendant types. Entity
id string The identifier for this item. Entity
meta object Metadata associated with this item. Entity

The Strong schema represents strongly emphasised content. It can contain any valid InlineContent nodes.

Examples

Simple

{
  "type": "Strong",
  "content": ["Some important information"]
}

Nested types

{
  "type": "Strong",
  "content": [
    "Some ",
    { "type": "Delete", "content": ["important"] },
    "essential information"
  ]
}
